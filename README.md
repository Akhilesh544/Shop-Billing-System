# 🛒 Shop Billing System

A desktop-based Shop Billing System developed using **Python**, **Tkinter**, and **SQLite**. The application helps manage products, generate bills, maintain sales history, and print customer receipts through an easy-to-use graphical interface.

---

## 📌 Features

- 🔐 Admin Login
- 📦 Product Management
  - Add Products
  - Edit Products
  - Delete Products
  - Search Products
- 🛒 Billing System
  - Add products to cart
  - Quantity selection
  - Remove items
  - Automatic total calculation
- 💳 Payment Window
  - Cash / UPI / Card
  - Balance calculation
- 🧾 Receipt Generation
  - Shop information
  - Customer details
  - Itemized receipt
  - Print Preview (PDF)
- 📊 Dashboard
  - Total Products
  - Total Bills
  - Total Revenue
  - Total Customers
- 📜 Sales History
  - View previous bills
  - Search bills
  - View receipt details

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Shop-Billing-System.git
```

Move into the project

```bash
cd Shop-Billing-System
```

Install dependencies

```bash
pip install reportlab
```

Create the database

```bash
python database.py
```

Run the application

```bash
python main.py
```
